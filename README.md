# 📚 Book Recommender System

An AI-powered Book Recommender System that suggests books based on user input using collaborative filtering. Built with Flask and a modern responsive UI.

---

## 🚀 Features

* 🔍 Search-based book recommendation
* 🤖 Collaborative filtering using cosine similarity
* 📊 Displays rating, votes, and book details
* ⚡ Fast and responsive web interface

---

## 🛠️ Tech Stack

* Python
* Flask
* NumPy
* Pandas
* Scikit-learn
* HTML + Tailwind CSS

---

## 🧠 How It Works

* User enters a book name
* System finds similar books using cosine similarity
* Recommendations are displayed with details (author, image)
